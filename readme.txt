A plugin allows to easily install a Talkerify.com widget on their wordpress website.

== Description ==

Talkerify for WP makes it easy for WordPress authors to offer live help/chat on their sites. Simply drop in the plugin, enter your Talkerify widget Id, and start chatting with your site visitors. No complicated settings, one file to drop in, and best of all it's FREE!

To learn more about Talkerify and how it works visit the official talkerify website https://www.talkerify.com/.

== Installation ==

1. Extract all files from the ZIP archive, making sure to keep the file structure intact.
2. Upload the `talkerify-for-wp` folder to the `/wp-content/plugins/` directory.
3. Activate the plugin through the `Plugins` menu in WordPress.
4. Go to the `Talkerify for WP` menu which is located under the `Settings` menu in the admin interface.
5. Enter your Talkerify widget id and then go to the frontend of your website to see your new chat widget in action!
